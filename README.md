# samalexanders.com

A lightweight, responsive personal research website for Sam Alexanders (MohammadReza EskandariNasab).

## Preview locally

From this folder, run:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Publish

This is a static website and can be deployed directly to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any standard web host. Point `samalexanders.com` to the selected host and upload the contents of this folder.
